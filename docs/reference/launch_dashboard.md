# Launch Risk Explorer Dashboard

Starts an interactive Shiny dashboard for exploring micromort and
microlife data.

## Usage

``` r
launch_dashboard(...)
```

## Arguments

- ...:

  Additional arguments passed to shiny::runApp()

## Value

Invisible NULL. Runs the dashboard until closed.

## Examples

``` r
if (FALSE) { # \dontrun{
launch_dashboard()
} # }
```

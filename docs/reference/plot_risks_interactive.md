# Interactive Risk Plot

Creates an interactive plotly visualization of risks with category
filtering.

## Usage

``` r
plot_risks_interactive(risks = common_risks())
```

## Arguments

- risks:

  Tibble. Dataframe of risks, defaults to
  [`common_risks()`](https://johngavin.github.io/micromort/reference/common_risks.md).

## Value

A plotly object with interactive filtering.

## See also

[`plot_risks()`](https://johngavin.github.io/micromort/reference/plot_risks.md),
[`common_risks()`](https://johngavin.github.io/micromort/reference/common_risks.md)

Other visualization:
[`plot_risks()`](https://johngavin.github.io/micromort/reference/plot_risks.md),
[`prepare_risks_plot()`](https://johngavin.github.io/micromort/reference/prepare_risks_plot.md),
[`theme_micromort_dark()`](https://johngavin.github.io/micromort/reference/theme_micromort_dark.md)
